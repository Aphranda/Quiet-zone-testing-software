本文档记录真实硬件环境下的手动验证项目。默认单元测试只覆盖 Mock 和纯逻辑；以下项目需要在连接 VNA、扫描架和开关箱的设备环境中执行。

## 环境准备

- 使用 uv 环境启动程序。
- 确认 VNA、扫描架、开关箱处于安全可控状态。
- 扫描架运动范围内无障碍物，急停和停止按钮可用。
- 先使用小范围扫描参数验证，再扩大范围。

## 连接验证

- VNA 真实连接成功，断开后 UI 状态恢复未连接。
- 扫描架真实连接成功，断开前能安全停止。
- 开关箱真实连接成功，断开后不保留错误状态。
- `连接全部` 任一设备失败时，已连接设备会被清理断开。

## VNA 验证

- 独立配置 sweep 参数成功。
- 独立采样返回指定点数的 trace。
- S 参数切换后采样结果标记正确。
- IF 带宽、功率、起止频率异常输入会被拒绝或报错。

## 扫描架验证

- 当前位置查询成功。
- X/Y 轴点动方向与 UI 标识一致。
- 绝对定位到小位移目标后位置接近预期。
- 相对定位后位置变化接近预期。
- 停止按钮能停止当前运动。
- 断开扫描架前会先尝试停止运动。

## 开关箱验证

- LCD74000F 或 TC500 profile 与实际设备型号一致。
- `S11/S21/S12/S22` 路由命令正确。
- 原始命令发送有响应或明确错误。
- 链路图显示与当前命令一致。

## 扫描流程验证

- 小范围步进扫描完成并生成 trace CSV、metadata、trace index。
- 小范围匀速扫描完成并生成 trace CSV、metadata、trace index。
- 扫描暂停后停在安全点，恢复后继续扫描。
- 扫描停止后返回已完成部分结果，不继续写入后续点。
- 扫描失败时 UI 恢复可操作状态，日志有错误记录。

## 验证记录

每次硬件验证建议记录：

- 日期和操作者。
- 设备型号、固件版本、连接方式。
- 扫描参数和输出目录。
- 通过/失败项。
- 异常日志和复现步骤。
