参数与符号约定
路损计算参数速览
本方案中校准结果最终进入路损数学计算。为避免符号方向混淆，先统一参数命名、单位和端口含义。
损耗参数
  • L_* 表示链路损耗/传输损耗，单位 dB
  • 符号由参数定义决定，不由名称决定
  • 含放大器链路可能为正值，表示净增益
辅助临时变量
  • L_AUX_* 表示辅助线缆损耗量，由原始 S21_AUX_* 转换得到
  • T_* 表示带辅助链路的临时闭合环路损耗
  • 二者都需保存，用于计算最终 L_*
原始测量
  • S21_* 表示网分原始 S21
  • 单位 dB，用于追溯和复算
  • 不直接替代 L_*

校准操作
校准模板
校准分为暗室内部链路、VNA 链路损耗、SA 校准、SG 校准四类；转台DUT到VNA的损耗作为 LINK-CAL-002 的 AUX-D 回传段一起完成。
LINK-CAL-001 暗室内部链路校准
  • 三根 3M 辅助线先独立测试
  • 辅助线直接接网分、暗室接口板和转台DUT接口
  • DUT-H/V 闭合测量包含馈源、反射面、标准增益喇叭
  • 再测 DUT 内部段；链路箱不下发命令
LINK-CAL-002 VNA 链路损耗校准
  • 主链路：转台DUT接口换标准增益喇叭
  • 三种工况：两边直通、AMP1 开启、AMP2 开启
  • H/V 到 VNA1；DUT 到 VNA2 可直通或经 AMP1
  • AUX-D 子步骤：校准转台DUT到VNA的损耗
LINK-CAL-003/004 SA 与 SG 校准
  • 校准仍用网分 S21，因此必须形成 PORT1→PORT2 环路
  • CAL-003：SA 接网分 PORT2，PORT1 侧经 VNA1→H/V 发射
  • CAL-004：SG 接网分 PORT1，PORT2 侧经 DUT→VNA2 接收
  • 两类校准分开执行
