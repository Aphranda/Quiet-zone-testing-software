# LCD74000F 指令文档

192.168.1.10:7

![image](./link.png)

### IEEE488.2

- `*IDN?`
  - ***查询板卡IDN信息***
- `*RST`
  - ***复位板卡，重置为默认配置***
- `*CLS`
  - ***清空错误队列***
- `*STB?`
  - ***查询status 寄存器状态***
- `*WAI`
  - ***同步指令，等待指令队列执行完毕***
- `*OPC?`
  - ***同步指令***
- `SYSTem:ERRor[:NEXT]?`
  - ***查询错误队列***
- `SYSTem:ERRor:COUNt?`
  - ***获取当前错误队列消息数量***

### SCPI

- `#`一般标识id号，如控制开关指令 `CONFigure:SWITCH#` ， 如果需要控制 `SW1` 则指令为 `CONFigure:SWTICH1 1`

| CMD                                                            | Summary                                  | value                        |
| -------------------------------------------------------------- | ---------------------------------------- | ---------------------------- |
| `CONFigure:LINK`                                             | 配置链路指令                             | [链路表](#链路表)             |
| `READ:LINK:STATe?`                                           | 查询链路状态                             |                              |
| `CONFigure:SWItch#`                                          | 配置单个开关的状态（调试用）             |                              |
| `READ:SWItch#?`                                              | 读取单个开关的状态（调试用），返回mask值 |                              |
| `CONFigure:SWItch#:MASk`                                     | 配置单个开关的状态,使用mask值（调试用）  |                              |
| ***以下指令非每个箱子都支持***                         |                                          |                              |
| `READ:DIP:SWItch#?`                                          | 读取拨码开关的值                         | [拨码开关指令](#拨码开关指令) |
| `CONFigure:MULSwitch# <in>,<out>`                            | 配置开关矩阵开关值                       |                              |
| `READ:MULSwitch#`                                            | 配置单个矩阵开关的状态（调试用）         |                              |
| `CONFigure:ATTENuator# <atten>`                              | 配置衰减器衰减值                         | 衰减值为浮点形式             |
| `READ:ATTENuator#`                                           | 获取衰减器当前配置                       | 衰减值为浮点形式             |
| `CONFigure:CANOPEN# <reg>,<value>`                           | 配置CANOPEN外设值                        |                              |
| `READ:CANOPEN# <reg>`                                        | 读取CANOPEN外设                          |                              |
| `TRIGger:STARt <optional trig name>,<optional trig segment>` | 启动触发                                 | [连续触发命令](#连续触发命令) |
| `TRIGger:STOP <optional trig name>`                          | 停止触发                                 | [连续触发命令](#连续触发命令) |
| `TRIGger:MANUal <optional trig name>`                        | 手动单次触发                             | [连续触发命令](#连续触发命令) |
| ***以下指令在链路箱V3系统中支持***                     |                                          |                              |
| `CONFigure:SYSLog:SERver <ip>`                               | 配置 syslog server ip地址                |                              |

#### canopen 外设指令

- `READ:CANOPEN1 AMP1_TEMP`
  - 读取canopen 寄存器 0x2000, 返回值： 浮点数
- `READ:CANOPEN1 AMP2_TEMP`
  - 读取canopen 寄存器 0x2001, 返回值： 浮点数
- `READ:CANOPEN1 AMP1_CURRENT`
  - 读取canopen 寄存器 0x2006, 返回值： 整型
- `READ:CANOPEN1 AMP2_CURRENT`
  - 读取canopen 寄存器 0x2007, 返回值： 整型
- `READ:CANOPEN1 AMP1_VOLTAGE`
  - 读取canopen 寄存器 0x200a, 返回值： 整型
- `READ:CANOPEN1 AMP2_VOLTAGE`
  - 读取canopen 寄存器 0x200b, 返回值： 整型
- `READ:CANOPEN1 AMP1_POWER`
  - 读取canopen 寄存器 0x200e, 返回值： 整型
- `CONFigure:CANOPEN1 AMP1_POWER,<value>`
  - 配置canopen 寄存器 0x200e,value: 整型
- `READ:CANOPEN1 AMP2_POWER`
  - 读取canopen 寄存器 0x200f, 返回值： 整型
- `CONFigure:CANOPEN1 AMP2_POWER,<value>`
  - 配置canopen 寄存器 0x200f,value: 整型
